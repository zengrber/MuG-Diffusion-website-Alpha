Model configurations

There are 4 model parameters available to be set by users:


  • 1.Sampling count: You can drag the button inside the slidebar to set the number of samplings. This decides how many charts MuG-Diffusion will generate.

  • 2.Sampling step: You can drag the button inside the slidebar to set sampling step. The meaning of this parameter is complicated to explained, it is recommended to adjust this parameter only when you know how this parameter works in stable diffusion.

  • 3.CFG Scale: You can drag the button inside the slidebar to set CFG scale. The meaning of this parameter is complicated to explain, it is recommended to adjust this parameter only when you know how this parameter works in stable diffusion.

  • 4.Random seed: You can input the number directly in the textbox to set the random seed. This influence the result of the random process during generating. You can set it to whatever you want.
