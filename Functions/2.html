LN Ratio settings

MuG-Diffusion supports two ways to set the ratio:


  • 1.Map type setting: Tick map type box to enable map type setting.
    3 map types is provided: Rice map means the LN ratio is set to a random number ranged from 0~10%. Hybrid map means the LN ratio is set to a random number ranged from 10%~40%. And LN map means the LN ratio is set to a random number ranged from 40%~100%.

  • 2.LN ratio setting: Tick ln ratio box to enable LN ratio setting.
    You can drag the button inside the slidebar or input number directly in the textbox to set the LN ratio.


Note that when both settings are enabled, you can have contradictory settings. Try to avoid this.
