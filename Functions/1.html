Difficulty settings

Supporting both osu! star rating system and Etterna MSD system.
Default setting is star rating system ranged from 1 to 8 stars. Tick MSD score box to enable MSD score setting. MSD score is ranged from 5.00 to 35.00. You can drag the button inside the slidebar or input number directly in the textbox to set the difficulty number. Note that higher star usually means higher MSD score. Try to avoid settings like "star rate=2, msd=32" as it is contradictory.
