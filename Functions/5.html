Model configurations

You can adjust the parameter of 7 patterns(according to MSD). For each type of pattern, Tick pattern name box to enable this setting.
After ticking the box, a corresponding radio box will appear. You can set more or less here. More means this pattern will appear in the map generated more frequently while less means the opposite.
Tick pattern name MSD box to enable MSD settings.
After ticking the box, a corresponding slidebar will appear. You can drag the butthon in the slidebar to set the MSD of this pattern. This will influence the output map's MSD of this pattern(calculated by MinaCalc).

Note that not ticking the box means allowing the machine to arrange this option freely. If you want to reduce specific patterns, please tick the box and select Less in the popup window.
