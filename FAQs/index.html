FAQ

Q: An error occurred!

Q: 报错了！

A: Common error solutions: try fixing the error based on the error message. If the message is not understandable, follow these steps:

  • 1.Check if the browser is Chrome/Edge

  • 2.Check if the console is closed

  • 3.Check if the proxy is enabled

  • 4.check if there are any special characters in the map file and rename it with only English letters if unsure

If all the steps above have been checked and the error still occurs, try restarting the programme; If restarting also don’t work, please provide feedback in the QQ group (548470510).

A: 常见报错解决方案： 看报错的提示自行修复，如果看不懂，那么按照下面的顺序进行操作：

  • 1.检查浏览器是否为Chrome/Edge

  • 2.检查是否关闭了控制台

  • 3.检查是否开启了代理

  • 4.检查谱面文件中是否有特殊字符,不知道有没有就重命名成只含英文字母的

以上都做了依旧报错 则重启，重启无效再在群(548470510)里反馈


Q: How to use Mug-Diffusion on my mobile phone?

Q: 手机怎么使用Mug-Diffusion？

A: Under most circumstances, mobile devices are not able to run Mug-diffusion. Please use PC instead.

A: 手机暂时不能使用，请使用电脑。


Q: How can I play the maps Mug-Diffusion generated on Malody?

Q: Malody怎么玩到？

A: Malody supports .osz file, which means you can open files generated by Mug-Diffusion directly with Malody

A: Malody支持 .osz格式的谱面压缩包，直接使用malody打开生成的文件即可。


Q: I don't tick the Jumpstream box, why there is still lots of jumpstream patterns in the map?

Q: 切换没打勾为什么还是有很多切？

A: Not ticking the box means allowing the machine to arrange this option freely. If you want to reduce specific patterns, please tick the box and select Less in the popup window.

A: 没打勾代表让机器自 由发挥，如果你想减少特定排列，请勾选后在弹出的框里选择 Less


Q: What will happen if I use contradictory parameters?

Q: 我使用自相矛盾的参数会怎么样？

A: There won't be any errors, but it is very likely that you won't get what you want.

A: 不会报错，但大概率不会生成你想要的东西。


Q: Can I provide my own maps to Mug-Diffusion in order to train it?

Q: 我可以自己喂谱给他训练吗？

A: This is temporarily not possible.

A: 暂时不行。


Q: Why the map doesn't contain any jacks?

Q: 为什么生成不出来叠？

A: If you are trying to generate a jack map with a high BPM, you have to change the parameters of minimal inverval(ms) for eliminating minijacks.

A: bpm太高的时候请同步更改 去子弹的参数(minimal inverval(ms) for eliminating minijacks.)数值。
